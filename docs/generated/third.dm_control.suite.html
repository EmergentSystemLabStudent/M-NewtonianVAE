third.dm_control.suite.build_environment(domain_name, task_name, task_kwargs=None, environment_kwargs=None, visualize_reward=False)

Returns an environment from the suite given a domain name and a task name.

Parameters

  • domain_name – A string containing the name of a domain.

  • task_name – A string containing the name of a task.

  • task_kwargs – Optional dict specifying keyword arguments for the task.

  • environment_kwargs – Optional dict specifying keyword arguments for the environment.

  • visualize_reward – Optional bool. If True, object colours in rendered frames are set to indicate the reward at each step. Default False.

Raises

ValueError – If the domain or task doesn’t exist.

Returns

An instance of the requested environment.
